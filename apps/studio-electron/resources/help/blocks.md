# Ein Block ist ein Würfel!

In Minecraft besteht fast alles aus Blöcken. Erde, Stein, Holz – alles Blöcke!

## Wie mache ich einen Block?

1. Klicke links auf das **+** Zeichen.
2. Wähle **"Neuer Block"**.
3. Gib ihm einen coolen Namen (z.B. "Super-Stein").
4. Male eine Textur (das ist das Bild auf dem Block).

Tipp: Du kannst einstellen, ob der Block leuchtet oder ob man durch ihn hindurchgehen kann!
